# 專注旅行計時器 🗺️

> 將你的專注時間化為台灣地圖上的旅行距離。邊工作、邊環遊台灣！

每小時 60 公里的虛擬速度，讓你在番茄鐘或正計時模式下，看著角色沿著真實公路一步步前進，抵達每個城市。

## 👉 [立即體驗](https://andyliuya.github.io/Focus-Journey/)

> 免安裝、免下載，點開就能用！建議使用 Chrome / Edge / Firefox 最新版。

## 快速上手

1. 開啟後選擇旅行路線，點擊「出發！」
2. 選擇 **正計時** 或 **番茄鐘** 模式
3. 點擊「開始旅行」，角色開始在地圖上移動
4. 隨時在左下角記錄待辦事項
5. 在右上角選擇喜歡的背景音樂
6. 結束後查看專注時間、移動距離，還能匯出成就圖！

## 個人化設定

開啟 `app.js`，修改頂部的常數即可客製化：

```javascript
// 角色頭像：可輸入 Emoji 或圖片網址
const USER_AVATAR = '🚶';

// 移動速度（公里/小時）
const SPEED_KMH = 60;
```

### 更換 BGM

在 `app.js` 的 `BGM_LIST` 陣列中替換音樂 URL：

```javascript
const BGM_LIST = [
    { name: '咖啡廳', icon: '☕', url: '你的音樂網址.mp3' },
    // ...
];
```

> 推薦免費音源：[Pixabay Music](https://pixabay.com/music/) / [Freesound](https://freesound.org/)

# 專注旅行計時器 🗺️

> 將你的專注時間化為台灣地圖上的旅行距離。邊工作、邊環遊台灣！

每小時 60 公里的虛擬速度，讓你在番茄鐘或正計時模式下，看著角色沿著真實公路一步步前進，抵達每個城市。

---

## 功能特色

| 功能 | 說明 |
|------|------|
| ⏱ 正計時 / 🍅 番茄鐘 | 可自訂工作、短休、長休時間，支援番茄計數 |
| 🗺 互動地圖 | MapLibre GL + CARTO Voyager 底圖，真實感十足 |
| 🚶 角色標記 | 沿 OSRM 真實公路路線移動，60fps 平滑動畫 |
| 🔒 視角鎖定 | 自動追蹤角色視角，手動拖曳後 10 秒自動拉回 |
| 🎵 BGM 播放器 | 內建 7 種環境音／白噪音，支援音量調整 |
| 📝 待辦事項 | LocalStorage 持久化，計時中追蹤完成任務數 |
| 💾 進度自動儲存 | 每 10 秒自動存檔，重啟後可選擇接續旅程 |
| 💤 閒置自動暫停 | 3 分鐘無操作自動暫停，回來後提示繼續 |
| 🧘 久坐提醒 | 可自訂連續操作閾值（預設 60 分鐘），超過時提醒休息 |

## 使用方式

### 直接開啟

無需安裝任何套件，直接用瀏覽器開啟即可：

```
index.html
```

> 建議使用 Chrome / Edge / Firefox 最新版。需要網路連線（地圖、路線規劃、BGM 皆為外部資源）。

### 快速上手

1. 開啟後選擇旅行路線，點擊「出發！」
2. 在計時器面板選擇 **正計時** 或 **番茄鐘** 模式
3. 點擊「開始旅行」，角色開始在地圖上移動
4. 在左下角的待辦事項隨時記錄任務
5. 在右上角選擇喜歡的背景音樂
6. 結束後查看本次的專注時間、移動距離、完成任務數

## 個人化設定

開啟 `app.js`，修改頂部的常數：

```javascript
// 角色頭像：可輸入 Emoji 或圖片網址
const USER_AVATAR = '🚶';

// 移動速度（公里/小時）
const SPEED_KMH = 60;
```

### 更換 BGM

在 `app.js` 的 `BGM_LIST` 陣列中替換音樂 URL：

```javascript
const BGM_LIST = [
    { name: '咖啡廳', icon: '☕', url: '你的音樂網址.mp3' },
    // ...
];
```

## 技術架構

- **前端**：純 Vanilla JS + HTML + CSS（無框架、無打包工具）
- **地圖**：[MapLibre GL JS](https://maplibre.org/) v3.6.2
- **底圖**：[CARTO Voyager](https://carto.com/basemaps/) (OpenStreetMap)
- **路線規劃**：[OSRM](http://project-osrm.org/) Public API（無 API Key）
- **字型**：Google Fonts — Noto Sans TC、Orbitron
